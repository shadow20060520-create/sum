### 3. 后端 API

| 接口 | 状态 | 实现详情 | 文件位置 |
|------|------|----------|----------|
| `GET /api/classes` | ✅ | 获取教师班级列表，支持Mock鉴权 | `api/classes/route.ts` |
| `POST /api/classes` | ✅ | 创建新班级，使用Service Role Key绕过RLS | `api/classes/route.ts` |

**API技术特点：**
- 使用 `SUPABASE_SERVICE_ROLE_KEY` 绕过RLS限制
- MVP鉴权：通过 `x-mock-user-id` Header获取教师ID
- 支持文本换行符切分（`\n` 和 `\\n`）

## 🗂️ 项目结构

```
sum/
├── supabase/                    # Supabase 数据库
│   └── migrations/              # 迁移脚本
│       ├── 0001_schema.sql      # 表结构定义（users, roles, classes, tasks, task_sentences, records）
│       └── 0002_rls.sql         # RLS策略配置
├── web-platform/                # 教师端 Next.js
│   ├── src/
│   │   ├── app/
│   │   │   ├── (auth)/login/    # 登录页（动态世界地图背景）
│   │   │   └── teacher/         # 教师工作台（班级管理、任务发布、学情看板）
│   │   └── api/                 # API接口
│   │       ├── classes/         # 班级管理（GET/POST）
│   │       └── tasks/           # 任务管理（GET/POST）
│   └── .env.local               # 环境变量（含真实Supabase密钥）
└── miniprogram/                 # 学生端小程序
    ├── pages/
    │   ├── login/               # 登录页（邮箱密码登录）
    │   ├── task-list/           # 任务列表（API获取+Mock降级）
    │   ├── practice/            # 练习页面（录音、评测、声调曲线）
    │   └── report/              # 报告页面（分数、视频）
    └── utils/
        ├── request.js           # HTTP请求封装
        ├── iflytek-ws.js        # 讯飞WebSocket管理器（完整框架）
        └── video-mapping.js     # 视频映射（1个真实视频+占位）
```
